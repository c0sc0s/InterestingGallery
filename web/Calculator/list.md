# 混沌计算器功能清单

## 📋 核心功能

### 基础计算
- 支持基本四则运算（+、-、×、÷）
- 支持百分比（%）
- 支持正负号切换（+/-）
- 支持小数点输入
- 支持键盘快捷键操作

### 随机答案生成
- 计算器不会给出正确答案，而是随机生成荒谬的答案
- 答案来自丰富的词汇库组合

---

## 🎨 视觉特效系统

### 1. 打字机效果
- **触发时机**：每次计算完成后
- **效果**：答案逐字显示，增加悬念感
- **特点**：
  - 随机打字速度（30-80ms/字符）
  - 打字过程中偶尔播放音效

### 2. 全屏烟花效果
- **触发时机**：计算完成时 15% 概率触发
- **效果**：在屏幕多个随机位置同时放烟花
- **特点**：
  - 5-9 个烟花同时爆炸
  - 每个位置有多个彩色烟花
  - 11 种不同颜色随机组合

### 3. 混沌视觉特效（随机触发，35% 概率）

#### 3.1 赛博故障（Glitch）
- 文字出现故障效果
- 多重阴影和错位动画

#### 3.2 反色效果（Invert）
- 整个页面颜色反转
- 背景变为浅色

#### 3.3 剧烈震动（Shake）
- 计算器剧烈震动
- 无限循环动画

#### 3.4 眩晕旋转（Dizzy）
- 计算器旋转 180 度并缩小
- 持续 2 秒

#### 3.5 重力崩坏（Gravity）
- 所有按钮（除等号外）掉落
- 随机旋转和位移
- 掉落后不可点击

#### 3.6 迪斯科模式（Disco）
- 背景颜色快速变化
- 红、蓝、绿、黑循环

#### 3.7 模糊效果（Blur）
- 计算器整体模糊
- 4px 模糊度

## 🔊 音效系统

### 按钮音效
- **触发时机**：每次点击按钮
- **特点**：
  - 随机音调（200-500Hz）
  - 短促音效（0.1 秒）

### 计算完成音效
- **触发时机**：计算完成时
- **特点**：
  - 上升音调序列（300→400→500→600Hz）
  - 每个音调间隔 50ms

### 彩蛋音效
- **触发时机**：触发彩蛋时
- **特点**：
  - C 大调和弦（261.63, 329.63, 392.00Hz）
  - 持续 0.3 秒

### 打字机音效
- **触发时机**：打字机效果过程中
- **特点**：
  - 10% 概率随机触发
  - 低音量音效（100-150Hz）

## 🎁 彩蛋功能

### 数字彩蛋

#### `42`
- **效果**：显示"生命、宇宙以及一切终极问题的答案"
- **视觉**：彩虹渐变文字 + 全屏烟花
- **音效**：彩蛋和弦

#### `404`
- **效果**：显示"404: 答案未找到"
- **视觉**：故障效果文字
- **音效**：彩蛋和弦

#### `666`
- **效果**：显示"恶魔的数字...但这里没有恶魔，只有混乱"
- **视觉**：红色文字 + 色相旋转滤镜（2 秒）
- **音效**：彩蛋和弦

#### `888`
- **效果**：显示"恭喜发财！但这里没有钱，只有虚无"
- **视觉**：黄色文字 + 全屏烟花
- **音效**：彩蛋和弦

#### `1337`
- **效果**：显示"LEET! 但你仍然得不到正确答案"
- **视觉**：绿色代码风格文字
- **音效**：彩蛋和弦

### 计数彩蛋

#### 计算 42 次
- **触发条件**：累计计算次数达到 42 次
- **效果**：显示"你已经计算了 42 次！这是巧合吗？"
- **视觉**：彩虹渐变文字

#### 计算 10 次后输入 0
- **触发条件**：计算次数 > 10 且输入为 0
- **效果**：显示"你计算了 X 次，但仍然一无所获"

### 时间彩蛋

#### 深夜模式
- **触发条件**：0 点或 23 点，10% 概率
- **效果**：显示"深夜模式：一切都没有意义"
- **视觉**：紫色文字 + 亮度降低（3 秒）

## 🎯 交互特性

### 键盘快捷键
- `0-9`：输入数字
- `.`：输入小数点
- `+`、`-`：加减运算符
- `*`：乘法运算符（显示为 ×）
- `/`：除法运算符（显示为 ÷）
- `Enter` 或 `=`：计算结果
- `Backspace` 或 `Escape`：清空

### 响应式设计
- 自适应字体大小
- 长文本自动换行
- 移动端触摸优化

### 状态管理
- 计算次数统计
- 打字机效果状态控制
- 特效状态重置机制

## 🎮 使用建议

1. **探索彩蛋**：尝试输入 `42`、`404`、`666`、`888`、`1337` 等数字
2. **连续计算**：计算多次看看会发生什么
3. **深夜使用**：在 0 点或 23 点使用可能有特殊效果
4. **观察特效**：注意计算完成后的随机视觉特效
5. **聆听音效**：打开声音体验完整的音效系统

## 💡 设计理念

> "Instrumental Rationality is Dead"
> 
> 这个计算器是对工具理性的解构，用荒谬和混乱来对抗对确定性的追求。每一个"答案"都是对意义的消解，每一个特效都是对秩序的破坏。
